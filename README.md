> "Crafting digital experiences that bend reality."

## 🚀 Key Features

*   **Liquid Glass Design System:** Real-time refraction effects and frosted-blur fallbacks for a stunning, tactile interface.
*   **Live Build Simulator:** A GSAP-powered terminal and browser mockup that simulates the real-time coding and building of a client website.
*   **Client-Side Order Gateway:** Complete checkout flow utilizing browser storage for a rapid, backend-free e-commerce experience.
*   **Interactive Data Visualizations:** Scroll-triggered, randomized metric counters that keep the interface feeling alive.
*   **Performance First:** Zero-dependency animations and optimized canvas rendering for buttery smooth 60fps interactions.

## 🛠 Tech Stack

*   **HTML5 / Semantic Structure**
*   **Vanilla CSS3** (Custom Properties, Flexbox/Grid, Liquid Glass styling)
*   **Vanilla JavaScript (ES6+)**
*   **Three.js** (WebGL 3D Spiral and Particles)
*   **GSAP (GreenSock)** (Advanced timeline animations and ScrollTrigger)
*   **Lenis** (Smooth Scrolling)
